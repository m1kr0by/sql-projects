## 📁 Проекты

### 1. Анализ продаж интернет-магазина
**Файл:** `01_store_analysis.sql`

**Данные:** `data.csv` (16 товаров, 4 категории)

**Что сделано:**
- Общая статистика по магазину (товары, выручка, категории)
- Анализ по категориям с расчётом долей
- Топ-2 товара в каждой категории по выручке
- Поиск "звёзд" продаж и аутсайдеров
- Ценовой анализ с динамикой (LAG/LEAD)
- Рекомендации по ценообразованию

**Инструменты:**
- SQL (SQLite)
- Оконные функции (ROW_NUMBER, LAG, SUM OVER)
- CTE (Common Table Expressions)
- CASE, агрегатные функции

**Как запустить:**
1. Импортируй `data.csv` в SQLite (или любой SQL редактор)
2. Запусти файл `01_store_analysis.sql`
3. Получи результаты анализа
